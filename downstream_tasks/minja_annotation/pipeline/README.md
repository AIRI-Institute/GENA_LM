\# Gene Finding Pipeline



This pipeline generates filtered transcript intervals (BED) from BigWig outputs produced by two evaluations on the same reference and chromosome:

\- 4-class evaluation: produces TSS/PolyA BigWigs and is used to build candidate transcript intervals.

\- 6-class intragenic evaluation: produces intragenic BigWigs and is used to filter candidate intervals.



Both evaluations use the same initial checkpoint:

\- `moderngena-base-pretrain-promoters\_multi\_v2\_resume\_ep30-ba90700`



The HuggingFace checkpoint is taken from:

\- `hf/`



\## Installation



```bash

conda env create -n bert24 -f bert24\_env.yml

conda activate bert24

python -m pip install -r requirements.txt

pip install "flash\_attn==2.6.3" --no-build-isolation

````



\## Validation: Generate BigWigs



Run evaluation twice (4-class and 6-class intragenic). Each evaluation writes a chromosome-specific output directory:



\* `.../eval/T2T-CHM13v2/<chromosome>/`



\### 1) 4-class evaluation (TSS/PolyA)



Config:



\* `configs/\_eval\_T2T\_chr20\_bidirectional\_modernGENA.yaml`



Run:



```bash

CUDA\_VISIBLE\_DEVICES=0 GENALM\_HOME=$(realpath ../../) \\

python evaluate\_on\_chromosome.py --config configs/\_eval\_T2T\_chr20\_bidirectional\_modernGENA.yaml

```



This command generates BigWigs in the output directory, including:



\* `tss\_+.bw`, `tss\_-.bw`, `tss\_+rev\_comp\_.bw`, `tss\_-rev\_comp\_.bw`

\* `polya\_+.bw`, `polya\_-.bw`, `polya\_+rev\_comp\_.bw`, `polya\_-rev\_comp\_.bw`



\### 2) 6-class intragenic evaluation (intragenic)



Config:



\* `configs/\_eval\_T2T\_chr20\_bidirectional\_modernGENA\_intragenic.yaml`



Run:



```bash

CUDA\_VISIBLE\_DEVICES=0 GENALM\_HOME=$(realpath ../../) \\

python evaluate\_on\_chromosome.py --config configs/\_eval\_T2T\_chr20\_bidirectional\_modernGENA\_intragenic.yaml

```



This command generates BigWigs in the output directory, including:



\* `tss\_+.bw`, `tss\_-.bw`, `tss\_+rev\_comp\_.bw`, `tss\_-rev\_comp\_.bw`

\* `polya\_+.bw`, `polya\_-.bw`, `polya\_+rev\_comp\_.bw`, `polya\_-rev\_comp\_.bw`

\* `intragenic\_+.bw`, `intragenic\_-.bw`, `intragenic\_+rev\_comp\_.bw`, `intragenic\_-rev\_comp\_.bw`



\## Configure the transcript interval script



Set paths to the evaluation output directories and the output BED file.



\### 1) 4-class BigWig directory (TSS/PolyA)



Variable:



\* `files\_path`



Value:



```python

files\_path = "/path/to/4class/eval/T2T-CHM13v2/<chromosome>/"

```



The directory is expected to contain TSS/PolyA BigWigs generated by the 4-class evaluation.



\### 2) 6-class BigWig directory (intragenic)



Variable:



\* `args\_bw\_dir`



Value:



```python

args\_bw\_dir = "/path/to/6class/eval/T2T-CHM13v2/<chromosome>/"

```



The directory is expected to contain intragenic BigWigs generated by the 6-class intragenic evaluation.



\### 3) Output BED path



Variable:



\* `args\_bed\_out`



Value:



```python

args\_bed\_out = "/path/to/output/transcripts\_filtered.bed"

```



Optional filtering parameters:



\* `args\_prob\_threshold`

\* `args\_zero\_fraction\_drop\_threshold`



\## Run



```bash

python3 pipeline.py

```



\## Output



Output file:



\* `args\_bed\_out`



BED columns:



\* column 1: chromosome

\* column 2: start (0-based)

\* column 3: end (0-based)

\* column 4: strand (`+` or `-`)

\* column 5+: preserved extra fields (if present)
